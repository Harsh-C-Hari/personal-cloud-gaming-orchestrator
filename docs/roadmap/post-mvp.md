# Post-MVP Vision

## Overview

The following ideas represent long-term possibilities for the platform.

These features are not part of the current MVP roadmap and should not influence current development priorities.

---

# Multi-Host Support

Potential capabilities:

* Multiple gaming hosts
* Host selection
* Host health routing
* Distributed orchestration

---

# User Management

Potential capabilities:

* Multiple users
* User profiles
* Save ownership
* User permissions

---

# Host Clustering

Potential capabilities:

* Clustered hosts
* Shared infrastructure
* Redundancy

---

# Advanced Save Management

Potential capabilities:

* Save versioning
* Save conflict resolution
* Differential backups
* Cross-host migration

---

# Advanced Monitoring

Potential capabilities:

* Hardware telemetry
* GPU monitoring
* Fan monitoring
* Temperature monitoring

---

# Advanced Recovery

Potential capabilities:

* Policy-driven recovery
* Self-healing workflows
* Predictive recovery actions
* More Tailscale state handling as additional failure states are discovered

---

# Cloud Expansion

Potential capabilities:

* Regional deployment
* Cloud-hosted infrastructure
* Hybrid host environments

---

# Marketplace Features

Previously discussed ideas such as:

* Public host sharing
* Marketplace systems
* Billing
* Payments

remain outside the current project scope and are not planned for the foreseeable future.

---

# Guiding Principle

The project remains focused on reliability, automation, and personal cloud gaming infrastructure before pursuing large-scale platform expansion.
